Unified Identity

    Enable millions or billions of your end users to use any login ID. Easily administer your employees and give everyone access to exactly what they need.

  • End users can use a single account with multiple logins to all customer web sites, apps, services
  • OAuth 2.0 Implementation and complete support for federated logins from 3rd party ID providers (Google, Facebook, Twitter, Sina, Tencent, Taboa/Alipay etc) in different markets, platforms & devices (incl. integration with Android, iOS, Windows accounts)
  • API for customer properties, apps & partners with centralized login and registration portal with iOS & Android SDK for client side integration. Very easy to compose and layer on customer specific APIs
  • Support multiple layers of identity including anonymous user, customer ID, Login ID for a specific property, family accounts, organizational accounts
  • Easy administration of end users and employees, customer support and partners
  • Authorization framework supports creation of multiple tiers of admins and privileged users & allows customer and partners to very easily provide multiple tiers of service to different users
  • Account security by adopting latest security features such as captcha, brute force detection, two factor authentication using SMS, velocity checking, One-time password (Application specific password).
  • High availability, low latency login & account management through GEO partitioning & replication over multiple data centers. Highly Scalable to billions of end users
  • Zero downtime on upgrades & Lots of experience doing Seamless, no down time migration from legacy systems

Authorization & Entitlements

Authorize every employee and end user for exactly what they should have. Enable each end user to have a large number of digital services

  • Flexible authorization system based on OAuth 2.0 protocol
  • Fine grain role based access control system so customer can define as many roles as needed with arbitrary fine grained access to APIs & functionality. Some examples:
    • Customer Support Personnel including Partners/Contractors
    • Partners: End Users, Software Engineers, Marketers
    • Advertisers: Administrator, Campaign Manager
    • Admin: Business Operations, Technical Operations, Developers
  • A highly scalable entitlement system for end users across multiple properties. This allows all of the customers properties to provide different services to the same customer
  • Online license management system (per device or per user) for software and downloadable content
